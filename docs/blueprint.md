# **App Name**: Type It!

## Core Features:

- Animated Text Display: Animate the given text line by line with a typewriter effect.
- Multi-line Input: Allow input of multiple lines of text to be displayed.
- Next Button Trigger: A 'Next' button to trigger the next line of animation, with optional auto-advance after a set time.

## Style Guidelines:

- Primary color: Electric purple (#8B3DFF) for an energetic coding feel.
- Background color: Dark grey (#28282B) to give focus to the vibrant primary color.
- Accent color: Bright blue (#3D8BFF) for the button to draw the user's eye.
- Font: 'Space Grotesk', a sans-serif, for a computerized look and readability.
- Code font: 'Source Code Pro' for accurate code snippet rendering.
- Simple, centered layout to emphasize the text animation.
- Use a typewriter effect for text reveal.
